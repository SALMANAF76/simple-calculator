# Aplikasi Kalkulator Sederhana
Sebuah aplikasi kalkulator sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript.

## Fitur:
- Operasi aritmatika dasar (+, -, ×, ÷)
- Tombol hapus (Backspace)
- Persentase (%)
- Kuadrat (x²)
- Kebalikan (1/x)
- Tampilan responsif dan desain yang bersih

## Cara Menjalankan
Cukup buka file index.html menggunakan browser.
